# Module template

Usar esta estructura para cada modulo nuevo:

```text
module-name/
  api/
  application/
  domain/
  infrastructure/
  tests/
  README.md
```

## README minimo

- Responsabilidad del modulo.
- Entidades principales.
- Casos de uso iniciales.
- Tablas o colecciones que toca.
- Eventos/contratos que expone.
- Cosas que no son responsabilidad del modulo.
