# Users module

Responsabilidad: usuario, perfil, carrera, preferencias y datos visibles en la app.

Incluye:

- Perfil de usuario.
- Carrera o contexto universitario.
- Preferencias de notificaciones y discovery.
- Relacion con comunidades y eventos a nivel de vista de usuario.

No incluye autenticacion tecnica; eso vive en `auth`.
